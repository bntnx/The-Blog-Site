¿Por Qué Existe Este Blog?

Este sitio web, **"Explorando el mundo"**, fue creado como un proyecto de simulación para demostrar nuestras habilidades y conocimientos en el uso de HTML y CSS.

  • Diseñar una estructura de páginas interconectadas.
  • Implementar componentes de diseño comunes como un menú de navegación fijo y un pie de página.
  • Aplicar estilos usando una única hoja CSS para mantener la consistencia.
  • Trabajar colaborativamente siguiendo una planificación SCRUM.
